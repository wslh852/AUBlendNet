## **Environment**
- Linux
- Python 3.6+
- Pytorch 1.9.1
- CUDA 11.1 (GPU with at least 11GB VRAM)

Other necessary packages:
```
pip install -r requirements.txt
```

## **Dataset Download**

The download link for the dataset is as follows:

- link: **https://pan.baidu.com/s/10pbmMknsFsFBm-FZEm5UXA?pwd=6vmd**

## **VQ-VAE Training and Testing**

Training/Testing Operations in Files:
```
python main/train_vq.py

python main/test_vq.py
```

### **AUBlendNet Training and Testing**

```
python main/train_pred.py

python main/test_pred.py
```


## **Interactive AU Editing**
You can achieve real-time AU control and emoji editing through the following script:
```
python main/au_control.py
```
